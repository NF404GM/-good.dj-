# good.DJ Design Language

## Product Character

`good.DJ` should feel like a studio instrument with stage confidence.

Desired traits:
- calm
- exact
- premium
- fast
- tactile
- authored

Undesired traits:
- loud
- gamer-ish
- derivative
- over-lit
- fake-industrial
- trend-chasing

## Visual Grammar

### 1. Structure Is Quiet

Most of the app should live in matte black, charcoal, warm gray, and off-white.
Panel seams, dividers, and structural labels should be present but understated.

### 2. Signal Is Rare

Signal color is reserved for:
- ready
- synced
- playing
- recording
- warnings
- stem identity

If everything glows, nothing matters.

### 3. Performance Data Feels Like Telemetry

Use mono text for:
- BPM
- key
- pitch
- dB
- timers
- state badges

Use Inter for:
- navigation
- labels
- copy
- controls

### 4. Tactility Without Cosplay

Controls should feel pressable and weighted, but not skeuomorphic.
Use border tension, subtle depth, restrained glow, and mass-aware motion.
Avoid decorative bolts, fake serials, or excessive “machine” labels.

## Signature good. Traits

- low-contrast structure with one strong active axis
- mono telemetry paired with warm interface text
- restrained emerald for live-ready states
- elegant center spine
- premium negative space
- motion that suggests mechanical confidence

## Copy Tone

Good copy is:
- direct
- calm
- technical when needed
- never theatrical

Examples:
- `READY`
- `SYNCED`
- `SEPARATING STEMS`
- `LOAD DECK A`
- `NO MODEL INSTALLED`

Avoid:
- “Let’s get the party started”
- “Magic mix”
- “AI-powered enhancement”
- “Oops!”

## Competitive Separation

Do not imitate:
- Serato’s gamer-performance aggression
- rekordbox’s utility density without warmth
- Traktor’s techno-futurist abstraction

Instead combine:
- editorial restraint
- instrument logic
- precise motion
- meaning-driven signal color

## Review Questions

When judging a surface, ask:

- Does this feel inevitable?
- Is the control hierarchy obvious?
- Is the surface calm under pressure?
- Does the app look more expensive because of restraint, not more decoration?
- Would a pro user trust this at first glance?
