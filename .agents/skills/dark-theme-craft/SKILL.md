# Dark Theme Craft

Read:

1. [AGENTS.md](C:\Users\tahme\OneDrive\Desktop\good.dj\AGENTS.md)
2. [docs/gooddj-design-language.md](C:\Users\tahme\OneDrive\Desktop\good.dj\docs\gooddj-design-language.md)
3. [src/styles/tokens.css](C:\Users\tahme\OneDrive\Desktop\good.dj\src\styles\tokens.css)

Use this elevation ladder:

- canvas
- surface-idle
- surface-active
- surface-raised
- overlay

Rules:

- Separate layers with subtle contrast first, borders second, glow last.
- Use signal glow only for active meaning: synced, live, recording, ready.
- Avoid pure black-on-black flattening.
- Avoid large colorful gradients unless they are structural and quiet.
- Prefer restrained seams, bevel tension, and depth over bright chrome.

Ask:

- Does this read premium because of restraint?
- Is the active state special enough without becoming loud?
- Is the surface still legible in a dark room?
